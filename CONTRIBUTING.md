# Contributing to Hermes Workspace

Student project. Contributions welcome.

## Ways to Contribute
- **Try it on your LLM.** Adapt INTERFACE.md for Claude/GPT/Qwen. Share your neural system table.
- **Replicate the experiment.** Run causal swap test on a different model. Report results.
- **Report bugs.** Open an issue.
- **Suggest improvements.** Architecture is a starting point, not a final design.

## Code Style
- Python: standard library preferred. Type hints encouraged.
- Config: markdown with consistent formatting.
- All contributions must pass sanitization check (no personal data).
